# Технология разработки программного обеспечения и оценка качества
# Курсовой проект

### Тема
Разработка автоматизированной системы "Система контроля и управления датчиками в производственном помещении" с полным жизненным циклом и оценкой качества.

### Цель работы
Систематизировать и применить на практике знания по технологии разработки программного обеспечения путем создания автоматизированной системы с полным жизненным циклом – от формирования требований до реализации и оценки качества готового продукта.

### Задачи:
- провести анализ предметной области и сформулировать требования к системе;
- разработать комплекс моделей системы на различных этапах жизненного цикла;
- реализовать функционирующий прототип системы;
- провести всестороннюю оценку качества разработанного ПО;
- оформить комплексную проектную документацию.

### Результаты работы:
1. Пояснительная записка с полным описанием проекта;
2. Комплект моделей ("IDEF0", "IDEF3" и "UML");
3. Работающий прототип системы;
4. Исходный код проекта;
5. Презентация для защиты;
6. Видео-демонстрация работы системы (2-3 минуты).
